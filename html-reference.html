HTML Element

Description

A valid HTML page must begin with a document declaration <!DOCTYPE html> at the beginning of the page to start coding.An HTML element usually consists of a start tag and end tag, with the content inserted in between:

Syntax

start tag: <tagname>

content: elements releated to the body

end tag: </tagname>

Example

<p>My first paragraph.</p>

Display

This is how an element looks in the browser

Head Element

Description

The tag <head> is a container for all the head elements (title, scripts, styles, meta information, and more)

Syntax

start tag: <head>

content: The Header

end tag: </head>

Example

<head>Header</head>

Display

The header is all of the titles, scripts, styles, and meta information.

Titles

Description

The <title> tag is required in all HTML documents and it defines the title of the document. The title displays a title for the page in search-engine results, provides a title for the page when it is added to favorites and defines a title in the browser toolbar.

Syntax

start tag: <title>

content: The Title

end tag: </title>

Example

<title>Title</title>

Display

This is the title

Body

Description

The <body> tag defines the document's body.The element contains all the contents of an HTML document, such as text, hyperlinks, images, tables, lists, etc.

Syntax

start tag: <body>

content: Body

end tag: </body>

Example

<body>Insert Body</body>

Headings

Description

HTML headings are defined with the tag <h1> to <h6> tags, where <h1> defines the most important heading and <h6> defines the least important heading of a page.

Syntax

start tag: <h1>-<h6>

content: Heading

end tag: </h1>-</h6>

Example

<h1>Insert Header</h1>

Display

This is a heading

Breaks

Description

The < br > tag inserts a single line break.The < br > tag is an empty tag which means that it has no end tag. These tabs are being used to separate these different sections.

Syntax

start tag: <br>

content: none

end tag: none needed

Links

Description

Links are found in nearly all web pages. Links allow users to click their way from page to page. HTML links are hyperlinks. You can click on a link and jump to another document.

Syntax

start tag: <a>

content: url

end tag: </a>

Example

<a>https://www.w3schools.com/html/html_links.asp</a>

Display

This is a link

Images

Description

Images can improve the design and the apperance of a web page. In HTML, images are defined with the <img> tag. The <img> tag is empty, it contains attributes only, and does not have a closing tag.

Syntax

start tag: <img>

content: image url

end tag: </img>

Example

<img>https://www.w3schools.com/html/html_links.asp</img>

Display

List Element

Description

An unordered list starts with the <ul> tag. The type attribute of the <ol> tag, defines the type of the list item marker: Each list item starts with the <li> tag.

Syntax

start tag: <ul><ol><li>

content: list information

end tag: </ul></ol></li>

Example

<ul>

<li>list information </li>

</ul>

Display

This is an example of an unordered list:
  • Coffee
  • Tea
  • Milk
This is an example of an ordered list:
  1. Coffee
  2. Tea
  3. Milk

Table Element

Description

An HTML table is defined with the < table> tag. Each table row is defined with the <tr> tag. A table header is defined with the <th> tag. By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

Syntax

start tag: <table><tr><td> <th>

content: table information

end tag: </table></tr></td> </th>

Example

<table style="width:100%"> <tr>

<th>Name</th>

<th colspan="2">Telephone</th>

</tr>

<tr>

<td>Bill Gates</td>

<td>55577854</td>

<td>55577855</td>

</tr>

</table></p>

Jill Smith 50
Eve Jackson 94

Blocks

Description

Every HTML element has a default display value depending on what type of element it is. The default display value for most elements is block or inline. A block-level element always starts on a new line and takes up the full width available (stretches out to the left and right as far as it can). The < div> element is a block-level element.

Syntax

start tag: <br>

content: none

end tag: none

Example

<div>Hello</div>

<div>World</div>
